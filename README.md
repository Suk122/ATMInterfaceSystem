#Description:-
This is a simple ATM System project developed using Java.
It simulates basic banking operations like balance checking, deposit, withdrawal, and PIN management.

#Features:-
Check account balance
Deposit money
Withdraw money
PIN verification and change
Multiple user support

#Technologies Used:-
Java
OOP (Object-Oriented Programming)
ArrayList

#How to Run:-
Open project in IntelliJ IDEA
Run the main class (ATM.java)
Follow on-screen instructions

#Output Example:-
User login with PIN
Menu options displayed
Transaction success messages

#Future Improvements:-
Add GUI (Graphical Interface)
Connect to database
Add transaction history
