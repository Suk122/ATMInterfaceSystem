# ATM Interface System (Java)

## Description
This is a console-based ATM simulation system developed using Java. It demonstrates core Object-Oriented Programming concepts and simulates basic banking operations.

## Features
- Account creation
- PIN authentication
- Balance inquiry
- Deposit money
- Withdraw money
- Multiple user support using ArrayList

## Technologies Used
- Java
- OOP (Classes & Objects)
- ArrayList
- Scanner (User Input)

## How to Run
1. Open project in IntelliJ / Eclipse
2. Compile the ATM.java file
3. Run the program
4. Follow on-screen instructions

## Learning Outcome
- Improved understanding of OOP concepts
- Learned how to manage multiple users
- Gained experience in building menu-driven applications
